## Problem Statement
Teachers and students need a quick way to record marks, compute percentages, and analyze class performance offline. This project implements a Student Result Analyzer using only Python.

## Scope
- CRUD-ish add and view of student marks.
- Local JSON storage and CSV export.
- Command-line interface.

## Target Users
Teachers, students, and academic staff.

## High-level Features
- Input marks per subject
- Compute totals and percentages
- Grade assignment and class summary
